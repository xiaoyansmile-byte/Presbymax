# 视力训练系统需求说明和定义文档

## 1. 项目概述

视力训练系统（ProsbyMax/MCL Vision Training）是一个基于网页的视力训练平台，旨在通过一系列专业的视觉训练项目，帮助用户提升视力适应能力、视觉注意力和视觉处理速度。

## 2. 核心功能需求

### 2.1 训练项目

系统包含以下训练项目：

| 训练项目 | 功能描述 |
|---------|--------|
| Gabor 配对 | 在网格中找出与金色提示格完全相同的另一格，训练视觉辨别能力 |
| 闪烁 Gabor 焦点锁定 | 在时序中快速判断目标空间频率，支持眨眼提示，训练视觉注意力 |
| 对比辨别 | 在细微差异中快速锁定更高对比的一侧，训练对比度辨别能力 |
| 阅读清晰度 | 尽快锁定可读信息，训练阅读时的焦点选择能力 |
| 眩光/散射可视性 | 在对比受损时仍选出更易辨识的一侧，训练在不良视觉条件下的识别能力 |
| 隧道远近切换识别 | 识别移动中的 E 字开口方向，强化远近焦点切换速度 |

### 2.2 系统功能

| 功能模块 | 功能描述 |
|---------|--------|
| 用户管理 | 管理用户信息和登录状态 |
| 训练计划 | 创建和管理训练计划，包括选择训练项目和设置训练时长 |
| 执行计划 | 按照预设计划执行训练，记录训练结果 |
| 数据分析 | 分析训练数据，生成训练报告 |
| 提醒设置 | 设置训练提醒，确保用户按时进行训练 |
| 数据同步 | 同步训练数据，确保数据安全 |
| 管理后台 | 管理 LocalStorage 中的训练数据，查看和删除历史数据 |

## 3. 非功能需求

### 3.1 响应式设计

- 系统应适应不同屏幕尺寸，包括桌面端和移动端
- 在移动设备上应提供良好的用户体验，确保训练项目能够正确显示和操作

### 3.2 设计风格

- 采用 Apple 风格的极简设计，包括浅色背景、简洁的布局和一致的视觉元素
- 保持整体风格统一，包括颜色、字体和交互效果

### 3.3 性能要求

- 页面加载速度快，确保训练项目能够流畅运行
- 训练数据存储在本地 LocalStorage 中，确保数据安全和快速访问

### 3.4 可用性

- 界面直观易用，操作流程简单明了
- 提供清晰的视觉反馈，帮助用户理解训练过程和结果

## 4. 数据需求

### 4.1 数据存储

- 训练历史数据存储在 LocalStorage 中
- 数据结构包括训练类型、得分、时长、最大难度等信息

### 4.2 数据管理

- 提供管理后台，允许查看和删除历史训练数据
- 支持训练数据的导出和同步

## 5. 业务流程

### 5.1 训练流程

1. 用户登录系统
2. 选择训练项目或执行预设训练计划
3. 进行训练，系统记录训练数据
4. 训练结束后，查看训练结果和分析报告
5. 系统保存训练数据到 LocalStorage

### 5.2 计划管理流程

1. 用户创建训练计划，选择训练项目和设置训练时长
2. 系统保存训练计划
3. 用户执行训练计划，系统按照计划顺序执行训练项目
4. 训练完成后，系统记录训练结果并更新计划状态

## 6. 技术要求

- 前端技术：HTML5, CSS3, JavaScript
- 数据存储：LocalStorage
- 响应式设计：媒体查询和弹性布局
- 图表库：Chart.js（用于数据可视化）

## 7. 项目范围

- 本项目为 Demo 系统，主要用于展示视力训练的核心功能
- 数据存储基于 LocalStorage，无需后端服务器
- 系统专注于训练功能的实现，不包含复杂的用户管理和数据分析功能
